ORQUESTACIÓN Y STACK DE OBSERVABILIDAD Infraestructura Cloud Security

Despliegue y correlación de métricas en entornos Linux multi-nodo sobre GCP.

Problema:

Ausencia de visibilidad centralizada sobre métricas de rendimiento de servicios empresariales distribuidos en la nube sobre distintas distribuciones Linux, impidiendo aislar cuellos de botella antes de que impacten en producción.

Solución:

Arquitectura multi-nodo en GCP con nodo central de observabilidad. Bastionado de permisos en directorios críticos (/opt), políticas estrictas de firewall y enrutamiento interno mediante VPC tags para comunicación segura entre nodos.

Capacidades:
Administración Linux Observabilidad (Prometheus/Grafana) Cloud VPC & Firewalls (GCP) Endurecimiento de Servidores
Tecnologías:

GCP, Prometheus, Grafana, Node Exporter, JMX Exporter, Apache Tomcat, Payara 6, systemd, Linux.

ECOSISTEMA CLOUD E INTEGRACIÓN IOT / AI — TEMPUS COFFEE IoT Cloud AI Security

Infraestructura cloud, pipelines de comunicación distribuida y consistencia de datos para hardware automatizado.

Problema:

Controlar de forma remota un sistema embebido en tiempo real, garantizando seguridad en la persistencia de datos y mitigando la naturaleza probabilística de los LLMs para evitar fallas en el hardware.

Solución:

Backend securizado con Sanctum. Sanitización avanzada de prompts antes de retransmitirlos. Orquestación de daemons persistentes para procesar ventanas críticas de tiempo real vía MQTT.

Lógica/Flujo:

Usuario/Voz → API REST → Sanitización → MQTT → Hardware

Tecnologías:

Laravel 11, Sanctum, MQTT, Gemini API, MySQL, Artisan CLI.

VIRTUAL CLIENT / AUTOMATIZACIÓN DE ENTORNOS VIRTUALIZADOS Linux Security

Abstracción y gestión de recursos a nivel de sistema operativo para hipervisores nativos de Linux.

Problema:

Riesgos de seguridad críticos de administrar hipervisores requiriendo elevación constante de privilegios root, o ejecuciones manuales ineficientes por consola.

Solución:

Sistema modular con interacción directa a la API de libvirt. Aislamiento de privilegios mediante grupos nativos del OS (libvirt / kvm), eliminando comandos root.

Tecnologías:

Python, PyQt, libvirt API, KVM, XML Architecture, Linux CLI.
